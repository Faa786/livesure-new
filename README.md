# LiveSure OP

Complete ERP + Delivery Management System for 19L Water Plants

## Features

- Authentication (Multi-role: Owner, Office Staff, Rider, Customer)
- Owner Dashboard with Analytics
- Office Dashboard with Order Management
- Customer Management with Ledger
- Rider Management with GPS Tracking
- Order Management with Status Tracking
- Inventory Management
- Bottle Tracking with QR Codes
- Subscription Management
- Expense Management
- Attendance System
- Franchise Management
- WhatsApp Integration (planned)
- Payment Integration Ready
- Reports & Analytics
- Real-time Notifications

## Tech Stack

- **Frontend**: Flutter
- **Backend**: Supabase
- **Database**: PostgreSQL
- **Authentication**: Supabase Auth
- **Storage**: Supabase Storage
- **Notifications**: Firebase
- **Maps**: Google Maps

## Installation

### Prerequisites
- Flutter SDK 3.0+
- Android Studio / VS Code
- Android SDK

### Steps
1. Clone the repository
2. Run `flutter pub get`
3. Create `.env` file with your credentials (see `.env` sample in repo)
4. Run `flutter run`

## Build APK

```bash
flutter build apk --release
```
